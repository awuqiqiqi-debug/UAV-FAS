## 二、FAS 流体天线（Fluid Antenna System）

### 2.1 定义

**类型：波束可重构离散端口流体天线（Beam-Reconfigurable Discrete-Port FAS）**

FAS 作为**唯一发射天线**搭载在无人机上，通过端口切换改变波束指向，通过标量增益控制发射功率。

### 2.2 物理结构

12 个离散端口在无人机机身上**水平线性排布**，间距 0.5λ ≈ 0.536cm，总长 ≈ 5.90cm。

```python
# src/envs/entity.py — UAV_BS_FAS
self.fas_port_spacing = 0.5 * 3e8 / 28e9  # 0.5λ = 0.00536m
self.fas_port_local = np.array([
    [self.fas_port_spacing * i, 0.0, 0.0] for i in range(-fas_num_ports // 2, fas_num_ports // 2)
])
```

### 2.3 端口选择机制

RL agent 输出 12 维 logits，通过 argmax 选择活跃端口：

```python
# src/envs/uav_comm_env.py — step()
port_logits = G[:self.UAV_BS_FAS.fas_num_ports]  # 12维
self.UAV_BS_FAS.fas_active_port = int(np.argmax(port_logits))
```

### 2.4 FAS 增益

FAS 增益为标量 F ∈ [1, 3]，由 RL 控制：

```python
self.UAV_BS_FAS.F = 1.0 + (G[self.UAV_BS_FAS.fas_num_ports] + 1.0) / 2.0 * 2.0  # [1, 3]
```

### 2.5 FAS 端口优化算法

`entity.py` 中保留了启发式端口选择方法，但当前环境中由 RL 直接控制：

```python
def fas_beam_optimize(self, ris_pos, eaves_pos):
    """遍历所有端口，按 metric = g_ris - g_eve 排序"""
    for idx in range(self.fas_num_ports):
        g_ris, _ = self.get_channel_gain(idx, ris_pos)
        g_eve, _ = self.get_channel_gain(idx, eaves_pos)
        metric = g_ris - g_eve
```

### 2.6 FAS 在信道模型中的体现

**活跃端口的综合信道**（标量）：

```python
# src/envs/uav_comm_env.py — calculate_comprehensive_channel_of_user_k()
active_port = self.UAV_BS_FAS.fas_active_port
h_d = np.asarray(self.h_U_k[k].channel_matrix)[active_port, 0]  # 直射路径
h_R_k = np.asarray(self.h_R_k[k].channel_matrix)  # RIS反射路径
h_UR_active = h_UR[:, active_port]  # 活跃端口的RIS信道
H_reflect = h_R_k @ amp_gains @ Phi_signal @ h_UR_active  # 级联反射
H_eff = (h_d + H_reflect) * self.UAV_BS_FAS.F  # × F增益
```

## 三、RIS 可重构智能反射面（Reconfigurable Intelligent Surface）

### 3.1 定义

**类型：多功能有源可重构智能反射面（Multi-functional Active RIS）**

核心特点：
- 每个反射单元有**放大增益** β ∈ [1, √11]
- 可**自适应分配**反射元件和干扰元件比例
- 独立控制**信号反射相位**和**人工噪声相位**
- 有**功率约束** P_RIS ≤ 30dBm (1W)

### 3.2 物理结构

8×8 均匀面阵（UPA），共 64 个反射单元，位于 (0, 50, 12.5)，法线朝 [0, 1, 0]。

### 3.3 有源放大增益 β

```python
BETA_MAX = 11.0  # β² ∈ [1, 11]
ris_beta = 1.0 + (ris_beta_raw + 1.0) / 2.0 * (np.sqrt(BETA_MAX) - 1.0)  # [1, √11]
```

### 3.4 自适应干扰比例

根据窃听者/用户信道强度比自适应分配反射和干扰元件：

```python
ratio = avg_eve_gain / (avg_user_gain + 1e-10)
jam_ratio = min(0.6, 0.3 + 0.3 * min(1.0, ratio))
jam_elements = int(self.RIS.ant_num * jam_ratio)     # 干扰元件数
reflect_elements = self.RIS.ant_num - jam_elements   # 反射元件数
```

| ratio | jam_ratio | 干扰元件 | 反射元件 |
|-------|-----------|---------|---------|
| 0 | 0.3 | 19 | 45 |
| 1 | 0.6 | 38 | 26 |
| >1 | 0.6 | 38 | 26 |

### 3.5 相位矩阵构建

信号反射和人工噪声分别占据不同元件：

```python
# 信号反射相位 (前 reflect_elements 个元件)
signal_expanded = np.tile(signal_phases, reflect_elements // 12 + 1)[:reflect_elements]
signal_full[:reflect_elements] = np.exp(1j * signal_expanded * np.pi)

# 人工噪声相位 (后 jam_elements 个元件)
jam_expanded = np.tile(jam_phases, jam_elements // 12 + 1)[:jam_elements]
jam_full[reflect_elements:reflect_elements+jam_elements] = np.exp(1j * jam_expanded * np.pi)
```

### 3.6 有源放大应用

```python
# 信号路径: β × Φ_signal → 放大用户信号
# 干扰路径: β × Φ_jam → 放大干扰噪声
self.RIS.Phi_signal = amp_diag @ Phi_signal_mat
self.RIS.Phi_jam = amp_diag @ Phi_jam_mat
```

### 3.7 RIS 功率约束

```python
P_RIS_MAX_dBm = 30  # 1W
P_RIS_total = P_reflect + P_jam + P_noise
if P_RIS_total > P_RIS_MAX_mW:
    scale = np.sqrt(P_RIS_MAX_mW / (P_RIS_total + 1e-10))
    self.RIS.Phi_signal *= scale
    self.RIS.Phi_jam *= scale
```

## 四、FAS 与 RIS 的协同工作

### 4.1 协同机制

```
FAS (无人机上)          RIS (地面固定)
    │                      │
    │  ① FAS 选择最优端口    │
    │  对准 RIS 方向发射     │
    │ ──────────────────→  │
    │                      │  ② RIS 反射元件增强用户信号
    │                      │  (β × Φ_signal)
    │                      │ ────────────→ 用户1 ✓
    │                      │ ────────────→ 用户2 ✓
    │                      │
    │                      │  ③ RIS 干扰元件干扰窃听者
    │                      │  (β × Φ_jam + 自适应比例)
    │                      │ ═══════════→ 窃听者 ✗
```

### 4.2 动作空间分配

```
Agent 1 总动作 (38维)：
├── [0:12]    FAS 端口选择 (softmax logits → argmax)
├── [12:13]   FAS 增益 F ∈ [1, 3]
├── [13:14]   RIS 放大增益 β ∈ [1, √11]
└── [14:38]   RIS 相位 (12信号反射 + 12人工噪声)

Agent 2 总动作 (2维)：
└── [0:2]    UAV 水平速度 (vx, vy)
```

## 五、信道模型

### 5.1 信道对象清单

| 信道 | 维度 | 路径损耗指数 | 链路类型 |
|------|------|------------|---------|
| h_U_k[k] | (12, 1) | n=3.5 | UAV→用户 直射 |
| h_U_p[p] | (12, 1) | n=3.5 | UAV→窃听者 直射 |
| h_UR | (64, 12) | n=2.2 | UAV→RIS |
| h_R_k[k] | (1, 64) | n=2.8 | RIS→用户 |
| h_R_p[p] | (1, 64) | n=2.8 | RIS→窃听者 |

### 5.2 有源RIS反射路径

$$H_{reflect} = h_R \cdot \text{diag}(\beta) \cdot \Phi \cdot h_{UR,active}$$

其中：
- $h_R$：RIS→用户/窃听者信道
- $\text{diag}(\beta)$：有源放大增益对角矩阵
- $\Phi$：RIS 相位矩阵（信号反射或人工噪声）
- $h_{UR,active}$：活跃端口的 UAV→RIS 信道

## 六、奖励函数

基于论文 Eq.(11) 的 tanh 归一化设计：

$$r = \tanh\left(\sum R_k^{sec} + 0.3 R_{fas} - p_m - 2p_r - p_e\right)$$

| 组件 | 含义 |
|------|------|
| $\sum R_k^{sec}$ | 保密速率之和（主目标） |
| $R_{fas}$ | FAS 端口保密增益 |
| $p_m$ | 功率约束惩罚 |
| $p_r$ | 最低安全速率惩罚 |
| $p_e$ | 能耗惩罚（仅在 ΣR^sec≥0 时生效） |
