## 二、FAS 流体天线

### 2.1 端口选择：Gumbel-Softmax Top-K

RL 输出 12 维 logits，通过 Gumbel-Softmax 实现可微分的离散选择，Top-K 同时激活多个端口：

```python
port_logits_tensor = torch.tensor(G[:12], dtype=torch.float)
if self.training:
    port_probs = F.gumbel_softmax(port_logits_tensor, tau=0.5, hard=False)
    _, topk_indices = torch.topk(port_probs, self.num_active_ports)  # K=2~3
else:
    _, topk_indices = torch.topk(port_logits_tensor, self.num_active_ports)
```

### 2.2 功率分配

K 个端口等功率分配，总功率受硬约束：

```python
F_total = 1.0 + (G[12] + 1.0) / 2.0 * 2.0  # [1, 3]
P_max_linear = 10 ** (30 / 10)  # 1000 mW
if F_total ** 2 > P_max_linear:
    F_total = np.sqrt(P_max_linear)
F_per_port = F_total / math.sqrt(K)  # 每端口功率
```

### 2.3 多端口信道容量

活跃端口的综合信道为各端口贡献之和：

```python
H_eff = 0
for port in active_ports:
    h_d = h_U_k[port, 0]  # 直射路径
    h_R_k @ amp_gains @ Phi_signal @ h_UR[:, port]  # RIS反射路径
    H_eff += (h_d + H_reflect) * F_per_port
```

## 三、RIS 有源反射面

### 3.1 放大增益 β

由 Agent 控制，映射到 [1, √11]：

```python
ris_beta = 1.0 + (ris_beta_raw + 1.0) / 2.0 * (np.sqrt(11.0) - 1.0)
```

### 3.2 干扰比例 η — Agent 控制

Agent 直接输出干扰比例，映射到 [0.1, 0.5]：

```python
jam_ratio = 0.1 + (jam_ratio_raw + 1.0) / 2.0 * 0.4  # [0.1, 0.5]
jam_elements = int(64 * jam_ratio)
reflect_elements = 64 - jam_elements
```

### 3.3 RIS 干扰相位控制（2026-06-23 更新）

**当前版本**：移除课程学习，Agent 100% 自主控制干扰相位。

```python
# 2026-06-23: 移除课程学习
self.jam_align_weight = 0.0    # 0%启发式，100% Agent自主
self.jam_align_decay = 1.0     # 不增长
self.jam_align_max = 0.0       # 不引入启发式对准
```

**原理**：课程学习让 Agent 只控制 50% 干扰相位，且收敛极慢。移除后 Agent 完全自主控制 RIS 干扰，探索效率更高。

### 3.4 功率约束

```python
P_RIS_total = P_reflect + P_jam + P_noise
if P_RIS_total > P_RIS_MAX:
    scale = sqrt(P_RIS_MAX / P_RIS_total)
    Phi_signal *= scale
    Phi_jam *= scale
```

## 四、信道模型

### 4.1 路径损耗

PL(dB) = 61 + 10·α·log₁₀(d) + Xσ

| 链路 | α | σ |
|------|---|---|
| UAV→RIS | 2.2 | 3 dB |
| RIS→用户/窃听者 | 2.8 | 3 dB |
| UAV→用户/窃听者 | 3.5 | 3 dB |

### 4.2 信道矩阵

H = e^(j·2π·f·d/c) · √PL · a_r · a_t^H

## 五、动作空间分配

```
Agent 1 总动作 (43维)：
├── [0:12]    FAS 端口选择 (Gumbel-Softmax logits → Top-K)
├── [12:13]   FAS 增益 F ∈ [0.3, 1.0]
├── [13:14]   RIS 放大增益 β ∈ [1, √20]
├── [14:15]   RIS 干扰比例 η ∈ [0.1, 0.5]
├── [15:27]   RIS 信号反射相位 (12维)
├── [27:39]   RIS 干扰相位 (12维)
└── [39:43]   用户级波束成形权重 (K×num_active_ports=4维)

Agent 2 总动作 (2维)：
└── [0:2]    UAV 水平速度 (vx, vy)
```
