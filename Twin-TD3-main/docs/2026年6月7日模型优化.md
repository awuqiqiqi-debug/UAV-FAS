## 三、动作空间详细说明

### 3.1 Agent 1 (FAS + RIS) — 38 维

| 参数范围 | 维度 | 含义 |
|---------|------|------|
| action_1[0:12] | 12 | FAS 端口选择 (softmax logits → argmax) |
| action_1[12:13] | 1 | FAS 增益 F ∈ [1, 3] |
| action_1[13:14] | 1 | RIS 放大增益 β ∈ [1, √11] |
| action_1[14:38] | 24 | RIS 相位 (12信号反射 + 12人工噪声) |

**代码实现**：

```python
# src/envs/uav_comm_env.py — get_system_action_dim()
def get_system_action_dim(self):
    result = 0
    if self.if_with_FAS:
        result += self.UAV_BS_FAS.fas_num_ports  # 端口选择: 12维 (softmax)
        result += 1      # FAS增益 F: 1维
        result += 1      # RIS放大增益 β: 1维
        result += 2 * 12  # RIS相位: 24维 (12单元×2)
    return result  # = 38
```

**端口选择**：RL 输出 12 维 logits，argmax 选择活跃端口。

```python
# src/envs/uav_comm_env.py — step()
port_logits = G[:self.UAV_BS_FAS.fas_num_ports]  # 12维
self.UAV_BS_FAS.fas_active_port = int(np.argmax(port_logits))

# FAS增益（标量）
self.UAV_BS_FAS.F = 1.0 + (G[self.UAV_BS_FAS.fas_num_ports] + 1.0) / 2.0 * 2.0  # [1, 3]
```

## 五、有源RIS模型

### 5.1 放大增益 β

```python
# src/envs/uav_comm_env.py — step()
BETA_MAX = 11.0  # β² ∈ [1, 11]
ris_beta = 1.0 + (ris_beta_raw + 1.0) / 2.0 * (np.sqrt(BETA_MAX) - 1.0)  # [1, √11]
```

### 5.2 自适应干扰比例

```python
# 窃听者信道强 → 多干扰元件，用户信道强 → 多反射元件
ratio = avg_eve_gain / (avg_user_gain + 1e-10)
jam_ratio = min(0.6, 0.3 + 0.3 * min(1.0, ratio))
jam_elements = int(self.RIS.ant_num * jam_ratio)
reflect_elements = self.RIS.ant_num - jam_elements
```

### 5.3 RIS 功率约束

```python
P_RIS_MAX_dBm = 30  # 1W
P_RIS_total = P_reflect + P_jam + P_noise
if P_RIS_total > P_RIS_MAX_mW:
    scale = np.sqrt(P_RIS_MAX_mW / (P_RIS_total + 1e-10))
    self.RIS.Phi_signal *= scale
    self.RIS.Phi_jam *= scale
```

## 六、奖励函数设计

基于论文 Eq.(11) 的 tanh 归一化设计：

```python
# src/envs/uav_comm_env.py — reward()
raw_reward = (total_secrecy            # Σ R_k^sec (保密速率之和)
              + 0.3 * R_fas            # FAS端口保密增益
              - p_m                    # 功率约束惩罚
              - 2.0 * p_r              # 最低安全速率惩罚
              - p_e)                   # 能耗惩罚 (Eq.12)
return np.tanh(raw_reward)             # 归一化到 [-1, 1]
```

| 组件 | 公式 | 说明 |
|------|------|------|
| 保密速率 | `Σ max(0, R_k - max(R_e))` | 论文主目标 |
| FAS 增益 | `0.3 × mean((g_user - g_eve) / g_user)` | 端口选择质量 |
| 功率惩罚 | `max(0, P - P_max) / P_max` | 超功率惩罚 |
| 安全速率惩罚 | `2.0 × Σ (R_th - R_k) / R_th` | 低于阈值时惩罚 |
| 能耗惩罚 | `0.1 × ΣR^sec × E_p_norm` | 仅在 ΣR^sec≥0 时生效 |
