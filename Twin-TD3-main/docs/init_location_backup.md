### 距离矩阵

| 链路 | 距离 |
|------|------|
| UAV → User1 | 45.0m |
| UAV → User2 | 45.0m |
| UAV → Attacker | 39.1m |
| UAV → RIS | 36.1m |
| RIS → User1 | 22.4m |
| RIS → User2 | 22.4m |
| RIS → Attacker | 11.2m |

### 设计理由

1. **对称分布**: User1和User2关于y轴对称，Agent学习更稳定
2. **RIS均衡覆盖**: 到两个用户距离相等（22.4m）
3. **窃听者威胁**: 在用户附近，构成真实安全挑战
4. **飞行效率**: UAV从中心出发，到各点距离适中
